CT1 (Sec A) - Q1

Draw the graph of the following function and find the domain and range of it. $$f(x) = \begin{cases} 0, & x \le -1 \\ \sqrt{1-x^2}, & -1 < x < 1 \\ x, & x \ge 1 \end{cases}$$

Step 1: Analyze the piecewise components

Interval 1 ($x \le -1$): $f(x) = 0$. This is a horizontal line along the x-axis from $-\infty$ up to and including $x = -1$.

Interval 2 ($-1 < x < 1$): $f(x) = \sqrt{1-x^2}$. Squaring gives $y^2 = 1 - x^2 \implies x^2 + y^2 = 1$. Since $y \ge 0$, this is the upper semi-circle of radius $1$ centered at the origin, excluding $x = \pm 1$.

Interval 3 ($x \ge 1$): $f(x) = x$. This is the line $y = x$ starting at $(1, 1)$ to $+\infty$.

Step 2: Domain

The function is defined for $x \le -1$, $-1 < x < 1$, and $x \ge 1$. The union covers all real numbers.

$$\text{Domain} = \mathbb{R} \quad (-\infty, \infty)$$

Step 3: Range

  • $x \le -1$: $y = 0$.
  • $-1 < x < 1$: $y = \sqrt{1-x^2} \in (0, 1]$.
  • $x \ge 1$: $y = x \in [1, \infty)$.
$$\text{Range} = \{0\} \cup (0, 1] \cup [1, \infty) = [0, \infty)$$

Step 4: Graph Description

  • $(-\infty, -1]$: Horizontal line $y = 0$, solid dot at $(-1, 0)$.
  • $(-1, 1)$: Upper semicircle of $x^2 + y^2 = 1$, open at $(\pm 1, 0)$.
  • $[1, \infty)$: Ray $y = x$, solid dot at $(1, 1)$.

Jump discontinuity at $x = 1$: left limit $0$, right value $1$.

CT1 (Sec A) - Q2

Write the conditions of continuity. Discuss the continuity of $f(x) = \begin{cases} x \cos \frac{1}{x}, & x \neq 0 \\ 0, & x = 0 \end{cases}$ at $x = 0$.

Step 1: Conditions of Continuity

A function $f(x)$ is continuous at $x = a$ if:

  1. $f(a)$ is defined.
  2. $\displaystyle\lim_{x \to a} f(x)$ exists.
  3. $\displaystyle\lim_{x \to a} f(x) = f(a)$.

Step 2: Check $f(0)$

From definition, $f(0) = 0$.

Step 3: Evaluate $\displaystyle\lim_{x \to 0} f(x)$

We need $\displaystyle\lim_{x \to 0} x \cos \frac{1}{x}$. Using the Squeeze Theorem: since $-1 \le \cos \frac{1}{x} \le 1$,

$$-|x| \le x \cos \frac{1}{x} \le |x|$$

As $x \to 0$, both $-|x|$ and $|x|$ approach $0$, so by the Squeeze Theorem:

$$\lim_{x \to 0} x \cos \frac{1}{x} = 0$$

Step 4: Compare limit and function value

$$\lim_{x \to 0} f(x) = 0 = f(0)$$

All three conditions are satisfied. Hence $f(x)$ is continuous at $x = 0$.

CT1 (Sec A) - Q3(i)

Find $\dfrac{dy}{dx}$ of $y = \sin^{-1}\left(e^{\tan^{-1} x}\right)$.

Let $u = e^{\tan^{-1} x}$. Then $y = \sin^{-1}(u)$.

$$\frac{dy}{dx} = \frac{1}{\sqrt{1-u^2}} \cdot \frac{du}{dx}$$

Now $\frac{du}{dx} = e^{\tan^{-1} x} \cdot \frac{1}{1+x^2}$. Substituting back:

$$\frac{dy}{dx} = \frac{1}{\sqrt{1 - e^{2\tan^{-1} x}}} \cdot \frac{e^{\tan^{-1} x}}{1+x^2}$$
Final Result:
$$\frac{dy}{dx} = \frac{e^{\tan^{-1} x}}{(1+x^2)\sqrt{1 - e^{2\tan^{-1} x}}}$$

CT1 (Sec A) - Q3(ii)

Find $\dfrac{dy}{dx}$ of $y = (\tan x)^{\cot x} + (\cot x)^{\tan x}$.

Let $y = u + v$ where $u = (\tan x)^{\cot x}$ and $v = (\cot x)^{\tan x}$.

For $u = (\tan x)^{\cot x}$:

Take $\ln u = \cot x \ln(\tan x)$. Differentiate:

$$\frac{1}{u}\frac{du}{dx} = -\csc^2 x \ln(\tan x) + \cot x \cdot \frac{1}{\tan x} \cdot \sec^2 x$$

Since $\cot x \cdot \frac{1}{\tan x} \cdot \sec^2 x = \csc^2 x$:

$$\frac{du}{dx} = (\tan x)^{\cot x} \csc^2 x (1 - \ln(\tan x))$$

For $v = (\cot x)^{\tan x}$:

Take $\ln v = \tan x \ln(\cot x)$. Differentiate:

$$\frac{1}{v}\frac{dv}{dx} = \sec^2 x \ln(\cot x) + \tan x \cdot \frac{1}{\cot x} \cdot (-\csc^2 x)$$

Since $\tan x \cdot \frac{1}{\cot x} \cdot (-\csc^2 x) = -\sec^2 x$:

$$\frac{dv}{dx} = (\cot x)^{\tan x} \sec^2 x (\ln(\cot x) - 1)$$

Combining:

Final Result:
$$\frac{dy}{dx} = (\tan x)^{\cot x} \csc^2 x (1 - \ln(\tan x)) + (\cot x)^{\tan x} \sec^2 x (\ln(\cot x) - 1)$$

CT1 (Sec A) - Q3(iii)

Find $\dfrac{dy}{dx}$ of $x^y = e^{x-y}$.

Taking natural log of both sides:

$$\ln(x^y) = \ln(e^{x-y}) \implies y \ln x = x - y$$

Solving for $y$:

$$y(\ln x + 1) = x \implies y = \frac{x}{1 + \ln x}$$

Differentiate using the quotient rule:

$$\frac{dy}{dx} = \frac{(1)(1 + \ln x) - x \cdot \frac{1}{x}}{(1 + \ln x)^2} = \frac{1 + \ln x - 1}{(1 + \ln x)^2}$$
Final Result:
$$\frac{dy}{dx} = \frac{\ln x}{(1 + \ln x)^2}$$

CT1 (Sec A) - Q4

State Leibnitz's Theorem. If $y = \sin(a \sin^{-1} x)$, show that $(1-x^2)y_{n+2} - (2n+1)xy_{n+1} + (n^2 - a^2)y_n = 0$ and find $(y_n)_0$.

Leibnitz's Theorem:

If $u(x)$ and $v(x)$ possess $n$-th order derivatives, then:

$$(uv)_n = \sum_{r=0}^{n} \binom{n}{r} u_{n-r} v_r$$

Step 1: Establish the base equation

Given $y = \sin(a \sin^{-1} x)$. Differentiating:

$$y_1 = \cos(a \sin^{-1} x) \cdot \frac{a}{\sqrt{1-x^2}}$$

Squaring and using $\cos^2\theta = 1 - \sin^2\theta$:

$$(1-x^2)y_1^2 = a^2(1-y^2)$$

Differentiating both sides and dividing by $2y_1$:

$$(1-x^2)y_2 - xy_1 + a^2y = 0 \tag{1}$$

Step 2: Apply Leibnitz's Theorem $n$ times

Differentiating (1) $n$ times using Leibnitz's theorem:

$$D^n[(1-x^2)y_2] - D^n[x y_1] + a^2 y_n = 0$$

For $(1-x^2)y_2$: let $u = y_2$, $v = 1-x^2$ ($v_1 = -2x$, $v_2 = -2$, $v_r = 0$ for $r \ge 3$):

$$D^n[(1-x^2)y_2] = (1-x^2)y_{n+2} - 2nxy_{n+1} - n(n-1)y_n$$

For $x y_1$: let $u = y_1$, $v = x$ ($v_1 = 1$, $v_r = 0$ for $r \ge 2$):

$$D^n[x y_1] = x y_{n+1} + n y_n$$

Substituting:

$$(1-x^2)y_{n+2} - 2nxy_{n+1} - n(n-1)y_n - x y_{n+1} - n y_n + a^2 y_n = 0$$
$$(1-x^2)y_{n+2} - (2n+1)xy_{n+1} + (a^2 - n^2)y_n = 0$$

This is equivalent to $(1-x^2)y_{n+2} - (2n+1)xy_{n+1} - (n^2 - a^2)y_n = 0$.

Step 3: Find $(y_n)_0$

At $x = 0$: $y(0) = 0$, $y_1(0) = a$, $y_2(0) = 0$.

Substituting $x = 0$ into the relation:

$$(y_{n+2})_0 = (n^2 - a^2)(y_n)_0$$

For even $n = 2k$: $(y_n)_0 = 0$.

For odd $n = 2k+1$:

$$(y_n)_0 = a(1^2 - a^2)(3^2 - a^2)\cdots((n-2)^2 - a^2)$$
Final Result:
$$(y_n)_0 = \begin{cases} 0, & n \text{ even} \\ a(1^2 - a^2)(3^2 - a^2)\cdots((n-2)^2 - a^2), & n \text{ odd} \end{cases}$$

CT1 (Sec A) - Q5

Find the $n$-th derivative of $y = e^{ax} \sin(bx+c)$.

Given $y = e^{ax} \sin(bx+c)$. Differentiating:

$$y_1 = e^{ax}[a \sin(bx+c) + b \cos(bx+c)]$$

Let $a = r \cos\phi$, $b = r \sin\phi$, so $r = \sqrt{a^2+b^2}$ and $\phi = \tan^{-1}(b/a)$. Then:

$$y_1 = r e^{ax} \sin(bx+c+\phi)$$

By induction:

$$y_n = r^n e^{ax} \sin(bx+c+n\phi)$$
Final Result:
$$y_n = (a^2+b^2)^{n/2} e^{ax} \sin\left(bx+c+n\tan^{-1}\frac{b}{a}\right)$$

ECE CT1 - Q1

Define one-one and onto functions with examples. Draw the graph and find the domain and range of $f(x) = |x-4| + |x| + |x+4|$.

One-One (Injective) Function:

$f: A \to B$ is one-one if $f(x_1) = f(x_2) \implies x_1 = x_2$. Example: $f(x) = 2x+3$.

Onto (Surjective) Function:

$f: A \to B$ is onto if $\text{Range}(f) = B$. Example: $f(x) = x^3$ for $f: \mathbb{R} \to \mathbb{R}$.

Analysis of $f(x) = |x-4| + |x| + |x+4|$:

Critical points at $x = -4, 0, 4$. Piecewise expansion:

$$f(x) = \begin{cases} -3x, & x < -4 \\ -x+8, & -4 \le x < 0 \\ x+8, & 0 \le x < 4 \\ 3x, & x \ge 4 \end{cases}$$

Domain: $\mathbb{R}$ (all real numbers).

Range: Minimum value is $f(0) = 8$. Function goes to $\infty$ at both ends.

$$\text{Range} = [8, \infty)$$

Graph: Symmetric about the y-axis. Line with slope $-3$ from $(-\infty, -4)$, then slope $-1$ to $(0, 8)$, then slope $+1$ to $(4, 12)$, then slope $+3$ to $\infty$. Vertices at $(-4, 12)$, $(0, 8)$, $(4, 12)$.

ECE CT1 - Q2

Define limit of a function. Test the continuity of $f(x) = \begin{cases} \dfrac{\sin^2 ax}{x^2}, & x \neq 0 \\ 1, & x = 0 \end{cases}$ at $x = 0$.

Limit Definition:

$\displaystyle\lim_{x \to c} f(x) = L$ if for every $\epsilon > 0$, there exists $\delta > 0$ such that $0 < |x-c| < \delta \implies |f(x) - L| < \epsilon$.

Continuity Test at $x = 0$:

$f(0) = 1$ (given). For $x \neq 0$:

$$\lim_{x \to 0} f(x) = \lim_{x \to 0} \frac{\sin^2 ax}{x^2} = \lim_{x \to 0} \left(a \cdot \frac{\sin ax}{ax}\right)^2 = a^2$$

For continuity, $\lim_{x \to 0} f(x) = f(0) \implies a^2 = 1 \implies a = \pm 1$.

If $a = \pm 1$, the function is continuous at $x = 0$. If $a \neq \pm 1$, it has a removable discontinuity at $x = 0$.

ECE CT1 - Q3(i)

Find $\dfrac{dy}{dx}$ of $y = \sin\left(2\tan^{-1}\sqrt{\dfrac{1-x}{1+x}}\right)$.

Let $x = \cos\theta$. Using half-angle identities:

$$\sqrt{\frac{1-x}{1+x}} = \sqrt{\frac{1-\cos\theta}{1+\cos\theta}} = \sqrt{\frac{2\sin^2\frac{\theta}{2}}{2\cos^2\frac{\theta}{2}}} = \tan\frac{\theta}{2}$$

Then:

$$y = \sin\left(2\tan^{-1}\left(\tan\frac{\theta}{2}\right)\right) = \sin\theta = \sqrt{1-x^2}$$

Differentiating:

$$\frac{dy}{dx} = \frac{d}{dx}\sqrt{1-x^2} = \frac{-2x}{2\sqrt{1-x^2}} = -\frac{x}{\sqrt{1-x^2}}$$
Final Result:
$$\frac{dy}{dx} = -\frac{x}{\sqrt{1-x^2}}$$

ECE CT1 - Q3(ii)

Find $\dfrac{dy}{dx}$ of $y = (\tan x)^{\sec x} + (\cot x)^{\csc x}$.

Let $y = u + v$ where $u = (\tan x)^{\sec x}$ and $v = (\cot x)^{\csc x}$.

For $u$:

$$\ln u = \sec x \ln(\tan x)$$
$$\frac{1}{u}\frac{du}{dx} = \sec x \tan x \ln(\tan x) + \sec x \cdot \frac{1}{\tan x} \cdot \sec^2 x$$
$$\frac{du}{dx} = (\tan x)^{\sec x} [\sec x \tan x \ln(\tan x) + \csc x \sec^2 x]$$

For $v$:

$$\ln v = \csc x \ln(\cot x)$$
$$\frac{1}{v}\frac{dv}{dx} = -\csc x \cot x \ln(\cot x) + \csc x \cdot \frac{1}{\cot x} \cdot (-\csc^2 x)$$
$$\frac{dv}{dx} = -(\cot x)^{\csc x} [\csc x \cot x \ln(\cot x) + \sec x \csc^2 x]$$

Combining:

Final Result:
$$\frac{dy}{dx} = (\tan x)^{\sec x} [\sec x \tan x \ln(\tan x) + \csc x \sec^2 x] - (\cot x)^{\csc x} [\csc x \cot x \ln(\cot x) + \sec x \csc^2 x]$$

ECE CT1 - Q3(iii)

Find $\dfrac{dy}{dx}$ of $x = y^{y^{y^{\dots}}}$ (infinite tower).

Since the tower is infinite, the exponent of $y$ equals the tower itself, which is $x$:

$$x = y^x$$

Taking natural log:

$$\ln x = x \ln y$$

Differentiate implicitly:

$$\frac{1}{x} = \ln y + \frac{x}{y}\frac{dy}{dx}$$

Solving for $\frac{dy}{dx}$:

$$\frac{dy}{dx} = \frac{y(1 - x \ln y)}{x^2}$$
Final Result:
$$\frac{dy}{dx} = \frac{y(1 - x \ln y)}{x^2}$$

ECE CT1 - Q4

State Leibnitz's Theorem. If $y = \cosh(m \cosh^{-1} x)$, show that $(x^2-1)y_{n+2} + (2n+1)xy_{n+1} + (n^2-m^2)y_n = 0$.

Leibnitz's Theorem:

$$(uv)_n = \sum_{r=0}^{n} \binom{n}{r} u_{n-r} v_r$$

Step 1: Establish the base equation

Given $y = \cosh(m \cosh^{-1} x)$. Differentiating:

$$y_1 = \sinh(m \cosh^{-1} x) \cdot \frac{m}{\sqrt{x^2-1}}$$

Squaring and using $\sinh^2\theta = \cosh^2\theta - 1$:

$$(x^2-1)y_1^2 = m^2(y^2-1)$$

Differentiating and dividing by $2y_1$:

$$(x^2-1)y_2 + xy_1 - m^2 y = 0 \tag{1}$$

Step 2: Apply Leibnitz's Theorem $n$ times

Differentiating (1) $n$ times:

$$D^n[(x^2-1)y_2] + D^n[x y_1] - m^2 y_n = 0$$

For $(x^2-1)y_2$: $v = x^2-1$, $v_1 = 2x$, $v_2 = 2$:

$$D^n[(x^2-1)y_2] = (x^2-1)y_{n+2} + 2nx y_{n+1} + n(n-1)y_n$$

For $x y_1$:

$$D^n[x y_1] = x y_{n+1} + n y_n$$

Substituting:

$$(x^2-1)y_{n+2} + 2nxy_{n+1} + n(n-1)y_n + xy_{n+1} + ny_n - m^2 y_n = 0$$
$$(x^2-1)y_{n+2} + (2n+1)xy_{n+1} + (n^2 - m^2)y_n = 0$$
